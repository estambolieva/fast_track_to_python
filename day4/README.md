Installation Requirements:  
1. pip  
   a. for Mac  
```javascript
brew install pip
```
   b. for Linux - download [get-pip.py](https://bootstrap.pypa.io/get-pip.py)  
```javascript
sudo python get-pip.py
```    
2. Flask  
```javascript
pip install flask
```  
3. MongoDB 
   a. including pymongo  
```javascript
pip install pymongo
```  
4. D3.js  
   a. download from [here](https://github.com/d3/d3/releases/download/v4.1.1/d3.zip)
